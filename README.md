# 📚 Library Management System

A simple Python-based Library Management System that uses a dictionary to store and manage book records.

## 📌 Features

- ➕ Add a new book with author name
- 📖 View all books in the library
- 🔍 Search for a book by name
- 🚪 Exit the application

## 🛠️ Technologies Used

- Python 3
- Dictionary
- Loops
- Conditional Statements

## 📂 Project Structure

```
Library-Management-System/
│── main.py
│── README.md
```

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run the Python file

```bash
python main.py
```
